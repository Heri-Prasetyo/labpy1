# Program menentukan bilangan terbesar dari 3 bilangan yang diinputkan <h1>

* Mulai
* Inisialisasi A, B, C
* Baca A
* Baca B
* Baca C
* Jika A > B dan A > C Maka Cetak A terbesar
* Jika tidak, B > A dan B > C Maka Cetak B terbesar
* Jika tidak, C Maka Cetak terbesar
